### backtesting 是很轻量的回测库，相比其它主流的库，它没有 indicator library，不用连接到 telegram 等。所以很轻量，也很易懂。是 event driven backtester。缺点是无法交易多资产等

### 安装

pip3 install backtesting
pip3 install ta-lib，非必需，只是此节教程需要。
