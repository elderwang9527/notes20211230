```
//标准库中的Add
pub trait Add<Rhs = Self> {
    type Output;

    fn add(self, rhs: Rhs) -> Self::Output;
}
```

默认泛型类型参数是 self，可省略不写，但一些情况下不能用默认的 self，就需要写了，trait 名尖括号里指定类型名。
